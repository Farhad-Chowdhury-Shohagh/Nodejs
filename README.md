# Node.js Beginner Projects

A collection of beginner-friendly Node.js projects covering command-line applications, file handling, HTTP servers, Express.js APIs, routing, JSON storage, and backend development fundamentals.

## About This Repository

This repository contains multiple Node.js practice projects created to improve JavaScript backend development skills. Each project focuses on a different concept of Node.js, such as user input, file system operations, HTTP servers, routing, REST APIs, and simple command-line applications.


## Technologies Used

- Node.js
- JavaScript
- Express.js
- npm
- JSON
- File System Module
- HTTP Module
- Readline Module

## Key Concepts Covered

- Node.js basics
- JavaScript runtime environment
- Command-line input and output
- File handling using `fs` module
- JSON data storage
- Creating HTTP servers
- Basic routing
- REST API development
- Express.js framework
- CRUD operations
- Callback functions
- Error handling
- User input validation
